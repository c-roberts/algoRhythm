Dependencies

  • Python 3.5+
  • Numpy
  • Librosa
  • Scipy
  • Lilypond

Backend


The user inputs an XML score and their performance of it.
The score is converted to the Lilypond format, which is easily parsable as text. The duration of notes is read, and the timesteps of the onsets are then derived based on the user's tempo and outputted into a Numpy arrray. The onsets of the user's performance are obtained using Librosa's onset detection module.
